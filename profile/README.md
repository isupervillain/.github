# isupervillain

> **Building AI agents that do real work.**

We're an early-stage AI lab building practical agentic tools for traders, farmers, and everyday operators. Not demos. Not wrappers. Agents that take action.

---

## What we build

Each agent is purpose-built for a specific domain — focused tools that understand their job and do it well.

| Codename | Agent | Status |
|---|---|---|
| **MIDAS** | Day Trader Agent — real-time market scanning, pattern recognition, risk management | `Experimental` |
| **DEMETER** | Farmer Agent — weather analysis, soil monitoring, crop health, yield optimization | `Planned` |
| **PROMETHEUS** | Research Agent — multi-source aggregation, intelligent summarization, trend tracking | `Experimental` |
| **HEPHAESTUS** | Workflow Agent — process automation, email triage, data pipeline orchestration | `Planned` |
